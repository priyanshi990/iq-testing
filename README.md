# DESCRIPTION 
This is a full-stack IQ Testing Website where users can register, log in, attempt dynamic IQ-based questions, and receive results with certificates. The platform includes a user dashboard, ranking system, admin panel for management, and an FAQ section for user support.

# HOW TO RUN
1. Install XAMPP  
2. Copy the project folder into `htdocs`  
3. Start Apache and MySQL  
4. Open phpMyAdmin and import the SQL file from the `database` folder  
5. Open browser and go to:  
   http://localhost/folder-name/i.php
